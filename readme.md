 
### Laravel simple Menu from data source

### Install
composer require begicf/simple-menu

### Migrate 
php artisan migrate

### How to use

Put in view blade template 
```
<x-menu-menu-component />
```
### Example  
 ``` html
<ul class="navbar-nav mr-auto">
  <x-menu-menu-component />
</ul>
 ```
